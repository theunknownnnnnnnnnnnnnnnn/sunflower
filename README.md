# sunflower

A static SSC CGL Preparation OS for tracking daily targets, weekly plans, syllabus coverage, editable chapters and topics, mock tests, revision flags, calendar progress, and history.

## Live app

This repository is configured for GitHub Pages through `.github/workflows/pages.yml`.

After the workflow finishes, the app should be available at:

https://theunknownnnnnnnnnnnnnnnn.github.io/sunflower/

## Local use

Open `index.html` in a browser. The app stores data in `localStorage`.
